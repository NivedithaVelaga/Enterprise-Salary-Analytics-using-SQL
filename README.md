# Enterprise Salary Analytics using SQL

## Project Overview

This project performs end-to-end workforce and salary analysis using advanced SQL.  
The objective is to analyze employee distribution, salary patterns, and organizational structure to generate actionable business insights.

This project demonstrates practical SQL techniques used in real-world HR and payroll analytics.

---

## Key Analysis Performed

### 1. Exploratory Data Analysis
- Total employee count
- Gender distribution
- Department-wise employee distribution
- Highest and lowest salaries
- Total payroll expense

### 2. Salary & Statistical Analysis
- Average salary by gender and department
- Salary spread using Variance and Standard Deviation
- Employees earning above/below department average
- Highest earners (company-wide and department-level)

### 3. Data Cleaning & Transformation
- Text standardization (UPPER, LOWER, TRIM)
- String extraction (SUBSTRING)
- Pattern matching (LIKE, SIMILAR TO)
- Numeric formatting (ROUND, TRUNC, CEIL, FLOOR)

### 4. Advanced SQL Concepts Used
- Subqueries & Correlated Subqueries
- Views for reusable logic
- INNER JOIN & LEFT JOIN
- GROUPING SETS, ROLLUP, CUBE
- Window Functions (RANK, ROW_NUMBER, LAG, LEAD, NTILE)

---

## Business Insights Generated
- Department salary comparison
- Executive salary analysis
- Regional workforce distribution
- Identification of missing division mappings
- Salary bucket classification (High / Middle / Low earners)
